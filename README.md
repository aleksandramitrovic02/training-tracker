# Training Tracker

A simple training tracking app.

## Tech Stack
- Backend: ASP.NET Core Web API, Entity Framework Core, SQL Server
- Auth: JWT + Refresh Token
- Frontend: Angular

## Repository Structure
- /backend  -> .NET solution (API + Core + Infrastructure)
- /frontend -> Angular app
- /docs     -> documentation, demo notes

## Features
- Registration / Login (JWT + refresh token)
- Workout logging (type, duration, calories, intensity 1-10, fatigue 1-10, notes, date & time)
- Progress tracking: monthly view grouped by weeks (total duration, workout count, avg intensity, avg fatigue)
